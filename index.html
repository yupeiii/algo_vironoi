Voronoi Diagram

M103140007 資安碩碩一 林郁珮

軟體規格書

輸入與輸出資料規格

輸入

  1. 以滑鼠在600*600的畫布上隨意點擊
  2. 點擊"file"中的"input file"讀入「輸入文字檔」與「輸出文字檔」

輸出

  1. 點選"File"中的"save file"來輸出「輸出文字檔」
  2. 輸出格式:

    "P" x y //原輸入點座標

    "E" x1 y1 x2 y2 //分割線段 (x1,y1)為起點 ，(x2,y2)為終點

    座標點以 lexical order順序排列（即先排序第一維座標，若相同，則再排序第二維座標；線段亦以 lexical order順序排列)

  3. 線段的lexical order (字典序)

    4. a. 線段E x1 y1 x2 y2，座標須滿足x1≦x2 或 x1=x2, y1≦y2

    b. 不同線段之間，依照x1, y1, x2, y2的順序進行排序(字典序)

  4. 範例如下:

    P 103 200

    P 193 64

    P 193 370

    P 283 200

    P 283 200

    E 0 34 193 161

    E 0 363 193 261

    E 193 161 193 261

    E 193 161 437 0

    E 193 261 600 476

功能規格與介面規格

介面

  1. 最上方為功能目錄欄
  2. 畫布上方之icon由左至右分別為 Run,Step by Step,Clean之功能
  3. 中間為視窗畫布，大小為600*600，其中左上方為原點(0,0)
  4. 右側顯示欄則為點擊畫布時的點座標
軟體測試規劃書

可以利用滑鼠直接點擊畫布，或者讀取輸入資料；並且使用直接執行即逐步執行之功能實現Voronoi Diagram於畫布上，來進行軟體測試。

  • 1~3點: 直接顯示
  • 4~6點: 一次Divide一次Merge
  • 7點以上: 多次Divde和多次Merge

詳細格式

!!請以UTF-8編碼

在檔案中、「#」符號開頭乃是註解及說明文字、可不予理會

3 // 總共有三個點

200 200 // 第1點座標(x, y) = (200, 200)

300 200 // 第2點座標(x, y) = (300, 200)

400 200 // 第3點座標(x, y) = (400, 200)

#中間也可加入備註

0 // 檔案結尾
